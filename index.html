Review Notes

  • Use this hub for one-session review; the source apps remain separated by responsibility.
  • Demo access roles: Park Guide, Admin, and Park Ranger are shown for tutor review. Current route access is demo-open; API role checks can be enabled with ROLE_CHECK_ENABLED=true.
  • The training platform is a seeded front-end demo; only AI/IoT monitoring incidents use the backend storage layer.
  • AI camera and IoT incident ingestion can require device tokens with DEVICE_TOKEN_AUTH_ENABLED=true.
  • Incident storage can run in memory mode for quick demos or MySQL mode using cos30049_assignment.
  • Direct links are useful if a dev server websocket does not work through the hub proxy.
